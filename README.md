# Terminator 2: Judgment Day — Landing Page

Статический сайт, посвящённый фильму «Терминатор 2: Судный день».
Проект создан в учебных и портфолио целях.

## Стек

- HTML5
- CSS3 (Grid, Flexbox, animations)
- JavaScript (ES6)
- Swiper.js

## Особенности

- кастомный аудиоплеер
- анимации при скролле (IntersectionObserver)
- доступность (aria-label, aria-hidden)
- секции актёров и съёмочной группы

## Локальный запуск

Откройте `index.html`
